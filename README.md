## ✨ Why HYUFA?

많은 대학생은:

- ✔ 재테크는 하고 싶은데, 어디서 시작해야 할지 모르고  
- ✔ 금융 용어가 어렵고  
- ✔ 앱은 복잡해서 계속 쓰기 어렵고  
- ✔ 누군가 대신 알려줬으면 좋겠고…  

그래서 HYUFA는 **“가벼운 첫 금융 습관”**을 돕는 **대화형 AI 서비스**로 설계됐습니다.

---

## 🧠 What HYUFA Can Do

| 기능 | 설명 |
|------|------|
| 💬 AI 금융 상담 | 소비 습관/목표 기반 개인화 조언 |
| 🔍 투자 성향 분석 | 간단한 질문으로 투자 타입 분류 |
| 📊 포트폴리오 추천 | 예금 → ETF 단계적 추천 |
| 🧮 계산기 | 적금·대출·금리 계산 |
| 📣 피드백 UX | 사용자 의견 → 다음 기능 개선 |
