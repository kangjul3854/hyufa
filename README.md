# 💙 HYUFA (HYU Finance Assistant)

**HYUFA**는 한양대학교 학생 및 사회초년생을 위한 **AI 금융 비서 서비스**입니다.  
금융 기초 상담부터 투자 포트폴리오 추천까지, 쉽고 친근한 대화형 상담을 제공합니다.

---

## 📌 서비스 정보
- **이름:** HYUFA  
- **의미:** HYU(한양대) + FINANCE + ASSISTANT  
- **목적:** 인간-인공지능 협업 제품 서비스 설계 수업 프로젝트  
- **형태:** 챗봇 기반 금융 상담 서비스

## 🧭 주요 화면
1. **#001 첫 화면** – “반갑습니다. HYUFA입니다.”  
2. **#002 개인정보 동의** – 상담 내용 수집·이용 안내  
3. **#003 메인 화면** – 날짜 / 인사 / 입력창  
4. **#004 메뉴** – 이용 안내, FAQ, 포트폴리오 추천, 계산기, 고객센터, 의견 남기기  
5. **#005~#010** – 각 기능별 세부 화면 구성  

---

## 💡 주요 기능
- 금융 기초 상담 및 용어 안내  
- 투자 성향 분석 후 포트폴리오 추천 (적금, ETF 등)  
- 적금·대출 계산기  
- 사용자 피드백 및 의견 수집  

---

## 🧠 비고
본 서비스는 상용 AI가 아닌 **프롬프트 기반 챗봇 프로토타입**으로,  
사용자 경험 검증 및 서비스 콘셉트 제시 목적입니다.
